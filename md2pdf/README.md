## markdown to pdf converter

This is a simple python script which converts a markdown file to a pdf file

First run `bash setup.sh` to initialize, you might need `python3-venv` and `pip3`packages and internet connection if you are running for the first time
After that source the enviroment using `source ./env/bin/activate` and run the script while in the enviroment:
```
  ./md2pdf input.md output.pdf
```
